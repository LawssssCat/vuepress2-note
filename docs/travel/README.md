### 医院

全国统一统一标准，不区别医院的背景和所有性质，按《医院分级管理标准》，根据医院规模、科研方向、人才技术力量、医疗硬件设施对医院资质进行评定后划分医院等级。

医院划分三级，每级再划分甲乙丙三等，其中三级医院增设特等。三级10等。
